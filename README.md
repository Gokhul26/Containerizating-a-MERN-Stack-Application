# A simple MERN stack application

**Note** - To run this project using `docker compose`, follow the below steps.


```
docker compose up
```


## Run it local without Docker

### Prerequisite

- Install `npm`

#### Start Server:

```
cd mern/backend
npm install
npm start
```

#### Start Client

```
cd mern/frontend
npm install
npm run dev
```
